FAQ

Câu hỏi về Game Development

EO làm được game thể loại gì?
Casual (hyper-casual, puzzle, arcade), mid-core (RPG nhỏ, strategy), sports simulation. EO không làm AAA 3D phức tạp. Sản phẩm tốt nhất của EO là mobile games đến mid-size indie.
Chi phí làm game từ bao nhiêu?
Hyper-casual prototype: 20-50 triệu. Mobile game đầy đủ (3-4 tháng): 80-200 triệu. Game có multiplayer + backend: 150-400 triệu. Giá phụ thuộc scope, số màn, hệ thống phức tạp.
Có hỗ trợ sau khi game ra mắt không?
Có. EO cung cấp dịch vụ LiveOps: thêm event theo mùa, cập nhật balance, fix bug, thêm nội dung. Gói retainer theo tháng hoặc thanh toán theo dự án nhỏ.
Bản quyền game thuộc về ai?
100% thuộc về khách hàng. EO bàn giao toàn bộ source code Unity, asset, backend code, database schema. Không giữ lại bất cứ thứ gì.
