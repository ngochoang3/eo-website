Chatbot AI đang chạy trên site này

🤖 EO Sales Chatbot

Icon chat ở góc dưới phải chính là sản phẩm AI của EO. Nó đọc toàn bộ knowledge base eo.io.vn và trả lời câu hỏi của khách hàng.

🔍 Kiến trúc RAG

Inverted index + TF-IDF, chạy offline-first trong trình duyệt. Không cần gọi API external — nhanh, không tốn token, private.

💡 Thử ngay

Hỏi thử: "EO có làm VPS không?" hoặc "Giá web development bao nhiêu?" — xem chatbot trả lời từ đúng knowledge base.

eo-chatbot-architecture
# User query pipeline
1. Normalize + tokenize query
2. Detect intent (FAQ/price/contact)
3. Search inverted index (TF-IDF)
4. Re-rank by semantic similarity
5. Template-based answer generation
6. Render with citations
✔ Response time: ~8ms
✔ Zero API calls
✔ Runs in browser

Chọn đúng model cho đúng bài toán

OpenAI GPT-4o

Reasoning phức tạp, code generation, multimodal (ảnh + text). Phù hợp chatbot enterprise yêu cầu cao.

Claude 3.5 / Gemini

FAQ

Câu hỏi về AI Solutions

Chi phí tích hợp AI từ bao nhiêu?
PoC cơ bản: 10-30 triệu. Chatbot production-ready: 30-80 triệu. RAG system với pipeline đầy đủ: 50-150 triệu. Phụ thuộc độ phức tạp, số lượng document, channel tích hợp.
Dữ liệu của tôi có bị gửi cho bên thứ 3 không?
Phụ thuộc model chọn. Nếu dùng OpenAI/Claude API — có. Nếu dùng Ollama on-premise — không, data không rời server của bạn. EO tư vấn chọn theo yêu cầu bảo mật dữ liệu của tổ chức.
AI có thể trả lời sai không?
Có. Không có AI nào đạt accuracy 100%. Với RAG, giảm hallucination vì AI trả lời từ tài liệu thực — nhưng vẫn cần human review với quyết định quan trọng. EO setup monitoring để phát hiện câu trả lời chất lượng thấp.
Tích hợp với hệ thống CRM/ERP hiện tại được không?
Được, qua API. EO kết nối AI với Salesforce, HubSpot, Odoo, SAP, hoặc hệ thống tự xây dựng. Webhook, REST API, hoặc database connector tùy case.
