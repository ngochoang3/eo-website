Cloud Ball

MMORPG Slime Fantasy Mobile — Unity · C# · Firebase · LiveOps

Unity C# Firebase LiveOps Mobile Game MMORPG

Slime Fantasy thế giới mở

Cloud Ball là MMORPG Slime Fantasy mobile do EO Technology tự phát triển với Unity. Game có 17 vùng đất rộng lớn để khám phá, hệ thống PvP đỉnh cao, cộng đồng Guild Bang Hội và hệ thống LiveOps cho phép cập nhật nội dung theo thời gian thực không cần update app.

Đây là sản phẩm game đầu tiên của EO Technology, chứng minh năng lực phát triển game mobile từ concept đến production, từ gameplay đến backend infrastructure và LiveOps.

Tính năng

Những gì Cloud Ball mang lại

🗺️

17 Vùng Đất

Thế giới mở rộng lớn với 17 biome khác nhau, mỗi vùng đất một hệ sinh thái Slime độc đáo và cốt truyện riêng.

⚔️

PvP Đỉnh Cao

Đấu trường real-time, ranked system, leaderboard toàn cầu. Thử thách kỹ năng với người chơi khắp nơi.

🛡️

Guild & Cộng Đồng

Hệ thống Guild đầy đủ — chiến tranh bang hội, kho bang, chat guild, boss chung, cống hiến.

🎯

LiveOps System

Sự kiện theo mùa, phần thưởng, nội dung mới cập nhật real-time không cần user update app.

Technology

  • • C# — Object pooling, ECS-lite

  • • Firebase Firestore (realtime DB)

  • • Firebase Cloud Functions
  • • Firebase Storage
  • • Remote Config (LiveOps)

📊 Analytics & Ops

  • • Firebase Analytics
  • • Crashlytics (crash reporting)
  • • LiveOps event system
  • • Admin dashboard
  • • CI/CD với GitHub Actions
