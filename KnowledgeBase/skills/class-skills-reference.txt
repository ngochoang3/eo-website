# Map skill 3 class — nguồn: F:\unitygame\skill.txt

## Unity setup
1. **Ball Skills → System Skill → Map All Class Skills From skill.txt** (loadout + skill assets)
2. **Ball Skills → System Skill → Setup Class Skill UI on Canvas** (HUD + panel)

Loadout:

- Skill assets: `Assets/script/Skill/Data/Skill_{Warrior|Archer|Mage}_*.asset`
- Combat: `Assets/script/Skill/Data/Combat/Combat_*.asset`
- Loadout: `Assets/Resources/GameplaySkills/Loadouts/Loadout_{Warrior|Archer|Mage}.asset`
- Runtime: `BallClassSkillBinder` gán 5 slot khi load class (`BallProgressController`)

Đủ cấp chưa quest → dấu `!` trên nút skill (UI — TODO).

## HUD
- Mana: thêm cột vào `ScoreHealthHudController` (field manaTextTmp).
- Panel skill: icon + tên + level + mô tả; khóa + mờ nếu chưa học (`ClassSkillProgressRules.IsSkillUnlocked`).

Code rules: `ClassSkillTxtCatalog`, `ClassSkillProgressRules`, `ClassSkillSlotMeta` trên loadout.
